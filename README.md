# StarInvaders🌏

**A retro-inspired, fast-paced space shooter built with Godot 4. Battle through waves of enemies, manage your defenses, and survive across multiple levels of increasing difficulty.**  

## Features
  - ***9 Challenging Levels: Progress through handcrafted stages with increasing enemy density and speed.***  
  - ***Three Unique Enemy Types: Each entity features distinct movement patterns and behaviors to keep you on your toes.***  
  - ***Original Soundtrack: Includes custom-composed music for both the Main Menu and the in-game action.***  
  - ***Classic Mechanics: Smooth movement, responsive shooting, and that quintessential "one-more-try" arcade feel.***

## Built With

- **Engine: Godot 4.x**  
- **Language: GDScript**  
- **Audio: BeepBox**  

## How to Play 

    Move: A/D or Left/Right Arrows
    Shoot: Spacebar
